Toast Notifications

Create

Notifications created using this form will use the standard settings set in the component.

You can also clear all notifications from with in the component by calling:

this._service.remove()

If you pass an id to the remove() method you can remove a single notification.

Push Notifications

Request Permission

Before you can create a push notification you need to ask for permission from the user.

Read more about this functionality here: Mozilla Developer network

this.pushService.requestPermission()

Create

Once you have permission you can send a push notification using the following method:

this.pushService.create('Just a title')
