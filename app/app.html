Angular2 Notifications Library

A light and easy to use notifications library for Angular 2.

Create a Notification

Notifications created using this form will use the standard settings set in the component.


                public options = {
                    timeOut: 5000,
                    lastOnBottom: true,
                    clickToClose: true,
                    maxLength: 0,
                    maxStack: 7,
                    showProgressBar: true,
                    pauseOnHover: true
                };
